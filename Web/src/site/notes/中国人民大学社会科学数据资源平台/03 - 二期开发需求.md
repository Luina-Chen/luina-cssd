# ⁠3. ⁠主内容区 - 数据预览
![image.png](https://cdn.jsdelivr.net/gh/Luina-Chen/CSSD/img/20260723220248283.png)
1. **功能概述：** 数据预览以表格形式展示数据内容，支持各列的<mark style="background: #ADCCFFA6;">排序</mark>与<mark style="background: #ADCCFFA6;">筛选</mark>功能，筛选方式根据列的数据类型自动适配。 #UI设计 
2. **表格基础功能：**
- 以表格形式展示数据内容，每列对应一个变量，每行对应一条观测值
- 各列支持单列排序：点击列头切换「升序/降序」三种状态
- 各列支持独立筛选：点击列头的筛选图标，弹出筛选面板，面板内容根据该列的数据类型自动适配
3. **各数据类型的筛选方式**： #UI设计 
![image.png](https://cdn.jsdelivr.net/gh/Luina-Chen/CSSD/img/20260723220548080.png)

【**ID列**】
- 自动识别变量名为 `id` 或 `ID` 的列
- 筛选方式：输入框，支持多个ID用逗号分隔（如 `1001, 1002, 1005`）

【**文本型**】
- 筛选方式：下拉选择匹配模式 + 输入框
- 匹配模式选项：等于 / 包含 / 开头是 / 结尾是
- 用户输入完整或部分内容进行精确或模糊匹配（如“张三”）

【**分类型**】
- 筛选方式：展示所有分类选项，以多选复选框（checkbox）形式呈现
- 选项上方提供搜索框，支持按选项名称快速筛选选项列表
- 每个选项旁显示该选项的统计数据百分比，如“男（52.0%）”
- 额外提供「缺失值」选项，用于筛选该列中为空值的记录

【**日期型**】
- 筛选方式：起始日期 + 结束日期
- 用户选择日期范围，筛选该范围内的数据

【**数字型**】
- 筛选方式：最小值 + 最大值
- 用户输入数值范围，筛选该范围内的数据

4. 筛选规则：
- 单个筛选条件选择后，在上方显示当前筛选，如`性别：男` ，显示筛选结果的观测值条数和总条数
- 各列筛选条件可叠加使用，多列筛选之间为 AND（且）关系
- 排序与筛选可同时生效

# ⁠4. ⁠主内容区 -变量概览
![image.png](https://cdn.jsdelivr.net/gh/Luina-Chen/CSSD/img/20260723221621759.png)
1. **功能概述：** 变量概览以表格形式展示文件内所有变量的元数据和统计摘要。 #UI设计 
2. **表格列定义**
- 变量名：变量在数据文件中的原始名称（如 `gender`、`age`）
- 变量标签：变量的中文描述性标签（如`性别`、`年龄`），无标签时显示「 — 」
- 数据类型：`数值` /` 分类` /` 日期`/`文本`
- 值标签：分类变量的编码与标签映射（如 `1=男, 2=女`），无标签时显示「 — 」
- 有效值个数：该变量非缺失值的记录数量
- 缺失率：缺失值占全部观测值的百分比，格式保留2位小数（如 `5.20%`），缺失率为0时显示`0%`
- 均值：数值型变量的算术平均值，保留2位小数；其余变量显示「 — 」
- 中位数：数值型变量的中位数，保留2位小数；其余变量显示「—」
- 分布预览：分类变量显示微型柱状图，分类变量按类别展示频数分布，柱子高度代表该区间/类别的频数或占比，其余变量不显示

【数据解析】以stata文件为例
Stata 文件（.dta）的数据和元数据存储在一起。解析时需同时读取这两部分。
![9ff5704d4a5366049ccefc1b791d5865.jpg](https://cdn.jsdelivr.net/gh/Luina-Chen/CSSD/img/9ff5704d4a5366049ccefc1b791d5865.jpg)
1. 解析时需提取每个变量的名称、标签、值标签映射、存储类型
2. 值标签映射需完整保留，用于数据预览展示分类变量时的可读文本
3. stata中数据类型有两个大类：数值型和字符型，数值型分为：byte、int、long、float、double
	- 判断变量的数据类型时，如变量在 Stata/SPSS 中定义了值标签（如 `1=男, 2=女`），无论底层存储类型是数值型还是字符型，判定为分类变量
4. 缺失值标记需识别，**需注意：** 数值型数据的缺失值是用点号 `.` 来表示的。它不是一个像字符型数据的空值（null），而是一个系统定义的“缺失”标识，并且在stata中会被视为一个无穷大的数值，确保统计指标（均值、缺失率等）计算准确
